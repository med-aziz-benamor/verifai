# Verifai 🔍🛡️
**See through the noise.** — AI-powered multi-modal content verification platform.

Verifai is a high-accuracy, multi-modal forensic platform that helps users detect misinformation, AI-generated media, deepfakes, and suspicious URLs in real-time. Built for journalists, investigators, and concerned citizens to verify the digital world with confidence.

Built for the **MenaCraft Hackathon 2026**.

---

## 🚀 Key Modules & Forensic Core

Verifai analyzes content across five categories and returns a structural forensic verdict with confidence scores, forensic signals, and **LLM-powered explanations**:

| Module | What It Detects | Core Engine |
|---|---|---|
| **Intelligence Feed** | Live monitoring of high-stakes viral trends | **Global OSINT + Forensics** |
| **Social Scraper** | Deepfakes & context matching from public URLs | **Axis-1 (ViT + CLIP)** |
| **Mail Scanner** | Phishing, context-hijacking, and attachments | **Groq Llama 3.3 + HF** |
| **Media Forensics** | Deepfake faces, GAN artifacts, and ELA heatmaps | **Ensemble ViT + CNN** |
| **URL Sandbox** | Phishing domains, threat intel, and SSL risk | **URLScan.io + Google** |

### 🛠️ Forensic Explainability (XAI)
Every analysis includes a **Forensic Summary** generated by Llama-3 or Gemini. Instead of just a "Fake" label, Verifai explains *why*—citing lighting inconsistencies, semantic mismatches between image and caption (CLIP), or specific GAN artifact patterns found in the pixels.

## ⚡ Quick Start

### 1. Backend (Python 3.11+)
```bash
cd backend
python -m venv venv
# Windows: .\venv\Scripts\activate | Unix: source venv/bin/activate
pip install -r requirements.txt
python -m uvicorn main:app --reload
```
*API Documentation: http://localhost:8000/docs*

### 2. Frontend (Node.js 18+)
```bash
cd frontend
npm install
npm run dev
```
*Dashboard typically runs on http://localhost:8081*

## 🔌 API Reference (v1)

| Method | Path | Description |
|---|---|---|
| **POST** | `/analyze` | Analyze local file or text |
| **POST** | `/analyze/post` | Scrape & analyze a public social media URL |
